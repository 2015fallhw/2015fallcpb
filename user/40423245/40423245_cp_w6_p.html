如何下載倉儲的資料


先打開start弄出y槽,leo editor(要自行下載),jupyter,SciTE及cmd等,處理好近端的作業環境。
cmd會預設y槽的路徑,即Y:\>,使用cd 來選擇自己要放作業的位置,例如我要放在tmp,就輸入cd tmp,就會進到tmp
之後複製在gh-pages的github clone URL,輸入:
git clone https...........
就能把倉儲的東西複製進y槽的tmp
但我們所在的分支是master，所以要切換到gh-pages
git checkout gh-pages
就會換成github gh-pages的檔案了
我們可以在裡面找到reveal.leo,這是我們要做作業用的檔案
下拉製作投影片,右拉上傳
每個指令碼都需要放在大於小於之間
section /section之間:個投影片
br /:換行
h數字:標題,數字為標題大小,內部建議用2 ,標題建議用1
會這些就能做一個基本的投影片了
git add -A>>拍快照
git commit -m"上傳資料名稱">>幫你的資料取個名字,方便於日後查看更新過程
如果第一次上傳或電腦有還原卡,會要求輸入信箱及使用者名稱
git push>>上傳
輸入完用戶名稱及密碼後,如果尚未更新,那可能是發生衝突,我們要先輸入git pull，更新最新檔案 進行除錯(刪除7個一行的大小等於,vr跟v的對應)後再上傳
